Cash Register Function

This page demonstrates a function that calculates the total price of items in a shopping cart.

Example:

var cartForParty = {
    banana: "1.25",
    handkerchief: ".99",
    Tshirt: "25.01",
    apple: "0.60",
    nalgene: "10.34",
    proteinShake: "22.36"
};

// Total: 60.55
